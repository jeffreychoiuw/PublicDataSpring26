# WA State Commuting Types 2018-2024 (D3 HW2)

## Description
This chart displays the change in commute types in Washington from 2018 to 2024. The most visible change is in 2021 due to the COVID pandemic, which shows a decrease in all forms of commuting but an increase in remote work from home. There is also a major gap between driving alone and using other forms of transportation. Using the D3 JavaScript library, I created six lines to represent the commute types. Lines, axes, and text are all formatted and contain comments to explain each code section.

Data Source: [U.S. Census Bureau](https://data.census.gov/table?q=public+transportation&g=040XX00US53&y=2024)
