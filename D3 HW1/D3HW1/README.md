# Top 15 Seattle City Departments by Employee Count (D3 HW1)

## Description
This chart displays the current most employed departments of the City of Seattle. Using the D3 JavaScript library, I created a horizontal bar chart. The margins and axes are formatted to fit the long department names and to show a clear difference between employee counts. The main.js code file contains comments to explain the purpose of each section.

Data Source: [City of Seattle Wage Data](https://data.seattle.gov/City-Administration/City-of-Seattle-Wage-Data/2khk-5ukd/about_data)
