# WA State Fatal Vehicle Crash Causes 2015-2025 (D3 HW3)

## Description
This chart shows the change in cause types of vehicle crashes in Washington from 2015 to 2025. These are some of the main factors that influence people's driving. A major change in the data is during 2020 where distracted driving or its legal reporting policy significantly decreases while drug usage increases. I used this [Custom Checkboxes and Radio Buttons](https://www.w3schools.com/howto/howto_css_custom_checkbox.asp) resource to create and style the checkboxes that link to my D3 data lines. I used this chart format in the previous D3 HW2 assignment but changed the data to match my upcoming Project 2 data.

Data Source: [Washington State Department of Transportation](https://wsdot.wa.gov/about/transportation-data/crash-data)
