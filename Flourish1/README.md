# Seattle Average Daily Traffic 2021-2026

## Description
This visualization displays the total traffic in Seattle for each day of the week. Each line represents the averages of a year from 2021 to 2026. The chart shows the rise in traffic volume since 2021 as well as the differences between weekdays, with Sundays as the lowest and Fridays as the highest.

The data is sourced from the [City of Seattle Open Data portal](https://data.seattle.gov/) and is provided by SDOT Traffic Counts Group. It contains 445k entries from 2010 to currently April 14, 2026, and was filtered to 2021-2026 for this visual. It was also filtered for a daily total greater than 5000 to remove errors, and then averaged based on weekday and entry year. The data source does not provide exact locations of data collection and likely does not represent the entire city of Seattle. 2026 is the only year without complete calendar year data but is still included for a current average.
